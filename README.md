# 📌 CRM System (C++)

## 🔹 Overview

The **CRM System** is a console-based application developed in **C++** that manages customer records efficiently. This project combines **Object-Oriented Programming (OOP)** with **Data Structures & Algorithms (DSA)** to simulate a real-world management system.

It allows users to perform core operations such as adding, updating, deleting, and searching customer data, while ensuring efficient storage and retrieval through optimized data structures.

---

## 🔹 Key Objectives

* To implement real-world problem solving using C++
* To apply OOP principles such as encapsulation, modularity, and abstraction
* To utilize DSA concepts for efficient data handling
* To build a structured and scalable management system

---

## 🔹 Features

* ➕ Add new customer records with validation
* ✏️ Update existing customer information
* ❌ Delete customer records
* 🔍 Fast search functionality
* 📁 Persistent storage using file handling
* 📊 Organized data management system

## 🔹 Data Structures & Their Role

### 📌 Binary Search Tree (BST)

* Used for storing customer records in sorted order
* Enables efficient searching, insertion, and deletion
* Reduces time complexity compared to linear search

### 📌 Linked List

* Allows dynamic memory allocation
* Stores records when size is not fixed
* Efficient insertion and deletion

### 📌 Stack

* Used for temporary storage and reverse operations
* Can be extended for undo/redo functionality

### 📌 Queue

* Maintains order of operations (FIFO)
* Useful for processing requests sequentially
