# 📌 CRM System (C++)

## 🔹 Overview

This project is a **Customer Relationship Management (CRM) System** developed in **C++** using Object-Oriented Programming concepts. It allows users to manage customer data efficiently through a simple console-based interface.

The system helps in organizing, storing, and retrieving customer information, making it useful for small-scale management systems and learning purposes.

---

## 🔹 Features

* ➕ Add new customer records
* ✏️ Update existing customer details
* ❌ Delete customer records
* 🔍 Search customers by different criteria
* 📁 File handling for permanent data storage
* 🧱 Structured using OOP (Classes & Objects)

---

## 🔹 Technologies Used

* C++
* Object-Oriented Programming (OOP)
* File Handling

## 🔹 How to Run

### 👉 Using Dev-C++

1. Open the project in Dev-C++
2. Press **F11 (Compile & Run)**

### 👉 Using Terminal

1. Open terminal in project folder
2. Run:
   g++ main.cpp -o crm
   ./crm
